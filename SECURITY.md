# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in actenon-scan itself (not a
finding in a scanned project — those are the tool working as designed),
please report it responsibly:

1. **Do not open a public GitHub issue.**
2. Email **security@actenon.dev** with a description of the vulnerability,
   reproduction steps, and impact assessment.
3. You will receive an acknowledgement within 48 hours.
4. We will work with you to verify and fix the issue before any public
   disclosure.

## Scope

This policy covers the `actenon-scan` tool itself — its parsing,
detection logic, CLI, and GitHub Action. It does not cover the code
that `actenon-scan` scans in your repositories (that is the tool doing
its job).

## Safe Harbor

We will not pursue legal action against security researchers who
responsibly disclose vulnerabilities in accordance with this policy.
